Welcome to MercuryLang â€” a free and open-source lightweight interpreter written in C++, inspired by the simplicity of Python. MercuryLang is designed to offer clean, readable, and expressive syntax, making it an ideal choice for developers of all experience levels.

Why MercuryLang?

  • Python-like syntax: MercuryLang has a syntax similar to Python, making it easy to learn and use.
  • Custom data types: MercuryLang supports custom data types, allowing you to create your own data structures.
  • Built-in functions: MercuryLang comes with a set of built-in functions for common tasks, making it easy to get started.
  • Simple error handling: MercuryLang has a simple error handling mechanism, making it easy to debug your code.
  • Lightweight: MercuryLang is lightweight and can be easily integrated into other projects.

Getting started

Before you download MercuryLang, make sure you have:

  • A computer with a modern operating system (Windows, macOS, or Linux)
  • A cmake tool to build from source, version 3.12 or higher
  • A C++ compiler (GCC, Clang, or MSVC)

Once you have these prerequisites, you can download MercuryLang from the link above. After downloading, follow the installation instructions in the README file.

Syntax

MercuryLang has a simple and easy-to-read syntax. Here are some examples:


# Hello World program in MercuryLang
!write("Hello, World!")

The code above prints "Hello, World!" to the console.

This is for variable name of variable dont start with number and only support alpha or "_"


# Variable declaration
LET x = 10
LET y = 20

This is for function and function call


# Function declaration
FUNC foo() DO
    !write("Hello from foo!")
END

# Function call start with '!'
!foo()

This is for math operator


# Math operations
LET a = 10
LET b = 20

a + b # Addition
a - b # Subtraction
a * b # Multiplication
a / b # Division

Comparison operator


# Comparison operations
LET a = 10
LET b = 20

a == b # Equality
a != b # Inequality
a > b  # Greater than
a < b  # Less than
a >= b # Greater than or equal to
a <= b # Less than or equal to

This is for if statement


# If statement
LET a = 10
LET b = 20
IF a > b THEN
    !write("a is greater than b")
END

Assignment expression


# Assignment expression
LET a = 10
a = a + 5

This is for while statement


# While loop
LET i = 0
WHILE i < 10 DO
    !write(i)
    i = i + 1
END
